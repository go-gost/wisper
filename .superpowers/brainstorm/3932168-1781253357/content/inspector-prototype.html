WebSocket Lifecycle

1. Open ws://{inspector}/api/records/tail?client_id=xxx&type=http
2. Each message = one HandlerRecorderObject
3. Prepend to list, cap at 200 records
4. On disconnect → auto-reconnect (1s→2s→4s→…→30s max)
5. On tab/protocol/type switch → close old WS, open new

Settings Inspector API Configuration

A new "Inspector" section in the Settings page where users configure the inspector API URL. When this URL is set, the "Traffic Inspection" button appears on tunnel/entrypoint detail pages. When empty, the button is hidden.
